# LAN Securty Concepts

### 1. Endpoint Security
    a. Network Attacks
       - Distributed Denial Denial of Service (DDoS)
       - Data breaches to steal confidential information.
       - Malware inifections like ransomware.
    b. Network Security Devices
       - VPN routers for secure remote access
       - Next-Generation Firewalls (NGFW)
       - Network Access Control (NAC) appliances like Cisco ISE.
    c. Endpoint Protection
       - Use of antivirus, AP, ESA and WSA.
       - Cisco ESA for email threat monitoring.
       - Cisco WSA for web-based threat mitigation.
### 2. Access Control
    a. AAA Framework
       Authentication: Control who accesses the network.
       Authorization: Define user actions and privileges.
       Accounting: Track user activities.
    b. IEEE 802.1X
       - Port-based access control for LANs
       - Roles: Client (Supplicant), Switch (Authenticator), and Authentication Server.
### 3. Layer 2 Security Threats    
    a. Switch Attack Categories
       MAC Table Attacks (e.g, flooding)
       VLAN Attacks (e.g., hoppping, double-tagging).
       DHCP Attacks (e.g., starvation, spoofing)
       ARP attacks (e.g., poisoning)
    b. Mitigation Techniques
       - Port Security, DHCP Snooping, DAI, and IPSG.
       - Secure management protocols: SSH, SCP, SFTP
